# Simple Linear Regression Salary Prediction

## Objective
The objective of this project is to predict salary values using Simple Linear Regression based on years of experience.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset Information
The dataset contains Years of Experience as the independent variable and Salary as the dependent variable.

## Project Steps
- Import Required Libraries
- Load Dataset
- Separate Feature and Target Variable
- Split Dataset into Training and Testing Sets
- Train Simple Linear Regression Model
- Predict Test Results
- Visualize Regression Line
- Compare Actual and Predicted Values

## Model Used
- Simple Linear Regression

## How to Run the Project
1. Install required libraries
2. Open the Jupyter Notebook
3. Run all notebook cells sequentially

## Output
The model predicts salary values based on years of experience and visualizes the regression line with actual data points.
