# Simple Linear Regression using Supply Dataset

## Objective
The objective of this project is to predict dependent values using Simple Linear Regression based on a single independent variable.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset Information
The dataset used in this project contains one independent feature and one target variable used for prediction.

## Project Steps
- Import Required Libraries
- Load Dataset
- Separate Feature and Target Variable
- Split Dataset into Training and Testing Sets
- Train Simple Linear Regression Model
- Predict Test Results
- Visualize Regression Line
- Compare Actual and Predicted Values

## Model Used
- Simple Linear Regression

## How to Run the Project
1. Install required libraries
2. Open the Jupyter Notebook
3. Run all notebook cells sequentially

## Output
The model predicts target values using Simple Linear Regression and visualizes the regression line with actual data points.
